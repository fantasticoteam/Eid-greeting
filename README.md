# 🌙 EidWish AI — Interactive Eid ul-Adha Greeting

An elegant, mobile-responsive web experience designed to generate and share beautiful, personalized Eid al-Adha greetings in seconds. Built entirely on the client side with a premium dark emerald and gold aesthetic, this application serves as a modern tool for sharing festive blessings across social platforms.

## ✨ Features

*   **Personalized Generation:** Instantly injects user names into custom greeting templates.
*   **Diverse Curation Styles:** Offers tailored messaging options ranging from traditional **Islamic** and high-end **Luxury** to modern **Gen-Z** and lighthearted **Funny** styles.
*   **Premium Visual Presentation:** Uses a smooth, responsive glassmorphism UI overlaying an abstract, glowing HTML5 Canvas backdrop.
*   **One-Click Image Download:** Leverages `html2canvas` to render the DOM element directly into a high-resolution `.png` file right in the browser.
*   **Native Share Integration:** Utilizes the Web Share API on mobile devices for seamless, single-tap sharing directly to WhatsApp, Telegram, or Instagram DMs.

---

## 🛠️ Tech Stack & Architecture

This is a **serverless, client-side application** designed to run with zero external server costs or database dependencies:

*   **Frontend:** Semantic HTML5, Vanilla JavaScript (ES6+), and fluid CSS3 transitions.
*   **Graphics:** HTML5 Canvas API managing an abstract ambient particle system.
*   **Libraries:** `html2canvas` (via CDN) for local, client-side client rendering of the downloadable card asset.

## 🚀 Deployment Instructions (GitHub Pages)

To launch this project live on your own GitHub profile:

1.  **Fork or Clone** this repository.
2.  Ensure your primary file is named exactly `index.html` inside the root directory.
3.  Navigate to your repository **Settings** ⚙️.
4.  Scroll down to the **Pages** section on the left-hand sidebar.
5.  Under *Build and deployment*, set the Source to **Deploy from a branch**.
6.  Select the **`main`** (or `master`) branch and click **Save**.
7.  Your custom link will be live at `https://yourusername.github.io/eid/` within 1–2 minutes!
